Each nested element uses a background one step deeper than its container.

Page — --bg-primary
Card — --bg-white
Nested section — --bg-secondary
Element — --bg-tertiary

Text Colors

--fg-secondary Secondary text for body content
--fg-tertiary Tertiary text for captions and metadata
